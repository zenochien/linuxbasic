Storage in Linux

Introduction to Storage Basic

List all Block devices

  • Block devices are special files that refer to or represent a device (which could be anything from a hard drive to a USB drive). So naturally, there are command line tools that help you with your block devices-related work.

  • Major Number is used to identigy the type of block device, value 8 represent a SCSI device starts with SD.

  1. Minor Number is uset to distuinguish individual, physical or logical devices.
$ lsblk 

$ ls -l /dev/ | grep "^b"
  1. To Print,Create and Delete the parition table use fdisk -l command 
$ sudo fdisk -l /dev/sda

Working with Ext4

  1. To create a file system we will make use of /dev/sdb disk, run below command
$ mkfs.ext4 /dev/sdb1
  1. Now create a directory to mount the filesystem use below commands
$ mkdir /mnt/ext4;

$ mount /dev/sdb1 /mnt/ext4
  1. To verify if the filesystem is mounted use
$ mount | grep /dev/sdb1

$ df -hP | grep /dev/sdb1

NFS

  1. NFS server maintains an export configuration file at /etc/exports that defines the clients which should be able to access the directories on the server. /etc/exports looks like this 
$ /etc/exports
/software/repos 10.61.35.201 10.61.35.202 10.61.35.203
  1. To exports all the mounts defined in /etc/exports use 
$ exportfs -a

$ exportfs -o 10.61.35.201:/software/repos

Working with Logical Volume Manager

  1. To make use of LVM, install the package LVM . 
$ apt-get install lvm2
  1. Use pvcreate command to create a Physical Volume.
$ pvcreate /dev/sdb
Physical volume "/dev/sdb" successfully created
  1. Use vgcreate command to create a Volume Group.
$ vgcreate caleston_vg /dev/sdb
Volume group "caleston_vg" successfully created

  1. To list the volume, you can use lvs command

  1. Now to create an filesystem you can use mkfs command
$ mkfs.ext4 /dev/caleston_vg/vol1
  1. To mount the filesystem use mount command
$ mount –t ext4 /dev/caleston_vg/vol1 /mnt/vol1
  1. Now logical volume is now available for use. Lets resize the filesystem on vol1 while it is mounted. Check the free space available.
$ vgs
