Working with Shell Basic

Working with Shell Basic

  1. Our goal is to create a directory structure, the top most directory which is /home/ec2-user/environment which is already created as it as a home directory but everything else underneath has to be created.

  1. To print the present working directory. Run pwd command
$ pwd
  1. To see the contents of the directory. Run ls command
$ ls
  1. To make (or) create a directory. Run mkdir command
$ mkdir Asia
  1. To make (or) create multiple directories. Run mkdir command followed by <directory_name1> <directory_name2> .. <directory_nameN>
$ mkdir Europe Africa America
  1. To change a directory from the current directory. Run cd <directory_name>
$ cd Asia
  1. To recursively created directories. Run mkdir -p <directory_name1>/<sub_directory_of_name1>
$ mkdir -p India/Mumbai
  1. To go back to one directory up. Run cd ..
$ cd ..
  1. To go back directly to a home directory of the current user from any location in the system. Run cd
$ cd

9. To change to a directory with absolute path. Run cd <directory_path>

$ cd /home/michael
  1. To Change to a directory with relative path. Run cd 
$ cd Asia

  1. To move file or directory. Run mv command
$ mv /home/ec2-user/environment/michael/Europe/Morocco /home/michael/Africa/
**or**
$ mv Europe/Morocco Africa/
  1. To rename a directory. Run mv command
$ mv Asia/India/Munbai Asia/India/Mumbai
  1. To copy a file to a directory. Run cp <destination_directorypath> command
$ cp Asia/India/Mumbai/City.txt Africa/Egypt/Cairo
  1. To delete a file from a directory. Run rm /path/ command
$ rm Europe/UK/London/Tottenham.txt
  1. To copy a directory recursively. Run cp -r command
$ cp -r Europe/UK Europe/UnitedKingdom
  1. To print the content of a file. Run cat /path/to/ command
$ cat Asia/India/Mumbai/City.txt
  1. To add a content to a file with cat(redirect) . Run cat > /path/to/ command
$ cat > Africa/Egypt/Cairo/City.txt
  Cairo
  `Type Ctrl + d from keyboard`
  1. To create an empty file. Run touch /path/to/filename command
$ touch /Asia/China/Country.txt
  1. To get the long list of files and directories. Run ls -l command
$ ls -l
  1. To list all files including the hidden. Run ls -la command
$ ls -a
  1. To list all the files in the order they were modified. Run ls -lt command
$ ls -lt
  1. To list all the files form oldest to newest. Run ls -ltr command
$ ls -ltr
  1. To check the home directory for a particular user say bob
$ grep bob /etc/passwd | cut -d ":" -f6
  1. To check the home directory for a particular user using built in shell variables
$ echo $HOME
  1. In the command echo Welcome, what does the word Welcome represent with respect to the command?
$ echo Welcome 
  - Where Welcome is an argument
  1. To check git command type
$ type git
  1. To create a directory
$ mkdir /home/ec2-user/environment/birds
  1. To create directories recursively
$  mkdir -p /home/ec2-user/environment/fish/salmon
  1. Create few more directories
$ mkdir -p /home/ec2-user/environment/mammals/elephant
$ mkdir -p /home/ec2-user/environment/mammals/monkey
$ mkdir /home/ec2-user/environment/birds/eagle
$ mkdir -p /home/ec2-user/environment/reptile/snake
$ mkdir -p /home/ec2-user/environment/reptile/frog
$ mkdir -p /home/ec2-user/environment/amphibian/salamander
  1. To move a directory
$ mv /home/ec2-user/environment/reptile/frog /home/ec2-user/environment/amphibian
  1. To rename a directory
$ mv /home/ec2-user/environment/reptile/snake /home/ec2-user/environment/reptile/crocodile
  1. To delete a directory
$ rm -r /home/ec2-user/environment/reptile

Bash environment variables

  1. To print SHELL environment variable
$ echo $SHELL
  1. To see a list of all environment variables. Run env from the terminal
$ env
  1. To set an environment variable with in the shell. The value is not carry forward to any other process.
$ OFFICE=caleston
  1. To set an environment variable we can use the export command. To make the value carry forward to any other process.
$ export OFFICE=caleston
  1. To persistently set an environment variable over subsequent login or a reboot add them to the ~/.profile or ~/.pam_environment in the users home directory.
$ echo "export OFFICE=caleston" >> ~/.profile 
**or**
$ echo "export OFFICE=caleston" >> ~/.pam_environment
  1. To check the value of a environment variable called LOGNAME
$ echo $LOGNAME

Path Variable

  1. To see the directories defined in path variable. Use the command echo $PATH.
$ echo $PATH
  1. To check if the location of the command can be identified. Use the which command Syntax: which <command>
$ which obs-studio
  1. To define a command in the PATH variable. To add we can use the export command.
$ export PATH=$PATH:/opt/obs/bin
$ which obs-studio

Customize Bash Prompt

  1. To see the value assign to PS1, type echo $PS1
$ echo $PS1
  1. To change the PS1 to only display the word ubuntu-server.
$ PS1="ubuntu-server"
$ echo $PS1

To customize further, have a look at the below special character.

  1. To change the bash prompt to display date, time, username of the current user, the hostname and the current working directory
$ PS1="[\d \t \u@\h:\w ] $ "
